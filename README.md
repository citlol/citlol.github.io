# Citlalli's Portfolio

A macOS-inspired personal portfolio website showcasing my projects, skills, and experience.

## Features

- Interactive terminal with custom commands
- Draggable desktop folders
- Dark/Light theme toggle
- Floating clouds and animated stars
- MySpace-style profile modal
- Responsive design for mobile and desktop

## Tech Stack

- React
- Vite
- CSS Animations

## Projects Showcased

- Pancake - Modern budgeting app (Co-Founder & Frontend Developer)
- Miel-Pomodoro - Productivity timer with animal pals
- Light Dal - Full-stack wishlist application
- Top Escrow Website Redesign - Contract work

## Terminal Commands

- `help` - List available commands
- `about` - Learn about me
- `projects` - View my projects
- `skills` - See my technical skills
- `contact` - Get my contact info
- `clear` - Clear terminal

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
